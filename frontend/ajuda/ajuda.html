O Painel Home é sua tela inicial e fornece uma visão "ao vivo" do status dos seus notebooks.

  • Cards de Status (Disponíveis, Emprestados, Indisponíveis): Mostram a contagem em tempo real de quantos notebooks estão em cada estado.
  • Gráfico de Barras: Mostra quantos empréstimos foram realizados em cada dia da semana (Segunda a Sexta).
  • Gráfico de Pizza: Mostra a proporção de notebooks em cada status, incluindo o total cadastrado.

Este módulo gerencia quem pode pegar notebooks emprestados.

1. Cadastrar Novo Usuário

  1. Vá em Usuários > Cadastrar Novo.
  2. Preencha todas as informações pessoais, acadêmicas e de contato. Campos com (*) são obrigatórios.
  3. O CPF e o Telefone possuem máscaras automáticas (ex: `XXX.XXX.XXX-XX`).
  4. O CEP busca o endereço automaticamente.
  5. Anexe os dois documentos obrigatórios (Declaração e Termo).

2. Buscar e Gerenciar Usuários

  1. Vá em Usuários > Buscar Usuário.
  2. Use os filtros de Nome, CPF, Curso ou Período para encontrar quem você procura.
  3. Ícone de Lápis: Permite editar as informações de um usuário, incluindo a substituição dos anexos.
  4. Ícone de Olho (Ativar/Desativar): Desativa um usuário, impedindo que ele apareça na lista de empréstimos.
  5. Ícone de Lixeira: Exclui um usuário permanentemente.
  6. IMPORTANTE: O sistema não permite Desativar ou Excluir um usuário que tenha um empréstimo ativo.

Este módulo gerencia os seus ativos (os notebooks).

1. Cadastrar Novo Notebook

  1. Vá em Notebooks > Cadastrar Novo.
  2. Preencha todos os campos.
  3. O sistema irá validar se o Nº de Tombamento ou o Nº de Série já existem, para evitar duplicatas.

2. Buscar e Gerenciar Notebooks

  1. Vá em Notebooks > Buscar Notebook.
  2. Use os filtros para encontrar um notebook específico.
  3. Status: Os cards mostram o status atual (Disponível, Emprestado ou Desativado).
  4. Ícone de Lápis (Editar): Permite alterar os dados do notebook.
  5. Ícone de Olho (Ativar/Desativar): Muda o status entre "Disponível" e "Desativado".
  6. Ícone de Lixeira (Excluir): Exclui o notebook permanentemente.
  7. IMPORTANTE: Um notebook com status "Emprestado" (ícone de cadeado) não pode ser editado, desativado ou excluído até que seja devolvido.

Este é o módulo principal para o dia-a-dia, onde você registra as saídas e devoluções.

1. Cadastrar Novo Empréstimo

  1. Vá em Empréstimos > Cadastrar Novo.
  2. A data e hora de retirada/devolução são preenchidas automaticamente (você pode alterar se precisar).
  3. No campo "Buscar Usuário", comece a digitar o nome ou CPF. Apenas usuários ativos e que não possuem outro empréstimo aparecerão.
  4. No campo "Buscar Notebook", comece a digitar o Tombamento ou Modelo. Apenas notebooks disponíveis aparecerão.
  5. Selecione os acessórios (ex: Fonte) e adicione observações, se necessário.

2. Empréstimos Ativos

  1. Esta tela mostra todos os notebooks que estão fora da instituição.
  2. Empréstimos que passaram da data/hora de devolução prevista ficam marcados em vermelho (Atrasado).
  3. Ícone de Check (✔) - Dar Baixa: Inicia o processo de devolução.
  4. Ícone de Lixeira: Exclui o registro de empréstimo. (Use com cuidado! Isso força o notebook a ficar "Disponível" sem registrar uma devolução).

3. Processo de Devolução (Dar Baixa)

  1. Ao clicar no Ícone de Check (✔), um modal de confirmação aparece.
  2. Você deve verificar se o notebook e os acessórios foram devolvidos em boas condições.
  3. Se o notebook estiver danificado, desmarque a caixa "Notebook devolvido em boas condições". Ao fazer isso, o campo "Observações" se torna obrigatório para você descrever o defeito.
  4. Clique em "Confirmar Devolução". O empréstimo sairá da lista de "Ativos" e irá para "Encerrados".

4. Empréstimos Encerrados

Esta tela é um histórico de todas as devoluções, mostrando quem pegou, quando pegou e quando devolveu, incluindo todas as observações de retirada e devolução (ex: se foi devolvido com defeito).
